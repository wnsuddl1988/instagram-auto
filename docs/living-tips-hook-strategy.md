## 2. LH-7 현재 hook 한계 분석

### 2-1. Scene 1 hook 원문
```
narration: "흰 옷 황변, 이 방법으로 5분이면 없어집니다!"
caption: "이거 실수" (→ caption_natural_v1에서 "이렇게 하지 마세요"로 개선)
```

### 2-2. 무엇이 부족한가

| 약점 | 설명 | 이유 |
|------|------|------|
| **소재 포화** | "흰 옷 황변"은 최소 수십 개의 기존 릴스가 이미 다룸 | 차별점 없음 |
| **후크가 결과 보장형** | "5분이면 없어집니다" — 시청자 반응: "그래서?" / "그건 다 아는 얘기" | 긴장감 없음 |
| **질문이 없음** | 시청자에게 "혹시 이거 잘못하고 있지 않나?" 라는 의심을 심지 못함 | 스크롤 멈춤 유발 실패 |
| **타깃이 불명확** | "흰 옷 있는 사람 모두"는 너무 넓음 → 개인화 없음 | 공감 연결 약함 |
| **긴박성 없음** | 오늘 당장 안 보면 손해라는 느낌 없음 | 저장/공유 유발 약함 |

```

✅ 강화: "흰 옷을 세탁기에 넣기 전에 이것만 안 하면, 또 누렇게 됩니다."
✅ 강화: "세탁 후에 흰 옷이 더 누렇게 되는 이유 아세요?"
✅ 강화: "흰 옷 망치는 세탁 실수 1위, 대부분이 지금도 하고 있어요."
```

## 3. Hook Pattern Library — 6가지 유형

생활꿀팁 릴스에서 스크롤을 멈추게 만드는 후킹 공식 6가지:

### 패턴 1: 금지형 (하지 마세요)
> "이렇게 하지 마세요" / "절대 하면 안 됩니다"

시청자에게 **자신이 잘못하고 있을 수 있다**는 의심을 심어 즉각 주의를 끈다.

```
예시:
- "냉장고 문칸에 달걀 넣지 마세요, 이유 있어요."
- "흰 옷, 세탁기에 그냥 넣으면 안 됩니다."
- "바나나를 냉장고에 넣는 분들, 당장 꺼내세요."
- "식빵을 냉동하는 방법이 잘못됐을 수 있어요."
```

**구조**: `[대상 오브젝트] + [흔히 하는 행동] + "하지 마세요/안 됩니다"` + 이유 암시

---

### 패턴 2: 손실형 (지금 이렇게 하면 손해)
> "그동안 돈 낭비한 거예요" / "이거 모르면 손해"

시청자가 **이미 손해를 보고 있다**는 손실 공포(loss aversion)를 자극한다.

```
예시:
- "과일 금방 상하는 이유, 지금 당장 확인하세요."
- "세제 이만큼 넣는 분들, 돈 낭비하고 있어요."
- "냄비 이렇게 닦으면 코팅 다 날아가요."
- "흰 옷 세탁할 때 이거 빼먹으면 매번 돈 낭비예요."
```

**구조**: `[행동/실수]` + `"손해/낭비/상해요"` + 긴박성

---

### 패턴 3: 반전형 (상식이 틀렸다)
> "사실 반대입니다" / "이게 더 좋아요"

시청자가 **알고 있다고 생각하는 상식이 틀렸다**는 인지 충격을 준다.

```
예시:
- "뜨거운 물이 더 잘 씻긴다? 흰 옷에는 반대예요."
- "세제를 더 넣으면 더 깨끗? 사실은 반대입니다."
- "냉장고 가득 채우면 신선도 유지? 반대예요."
- "기름 얼룩, 물로 먼저 씻으면 안 됩니다."
```

**구조**: `[흔한 상식/믿음]` + `"사실은 반대/틀렸어요"` + 호기심 자극

---

### 패턴 4: 타깃형 (특정 상황 공감)
> "이런 분들 꼭 보세요" / "혹시 이런 경험 있으세요?"

**특정 상황**을 직접 짚어 "나 얘기인가?" 반응을 유도한다.

```
예시:
- "세탁 후에도 흰 옷이 누런 분들, 이게 원인이에요."
- "음식을 금방 상하게 하는 냉장고 습관 있는 분들."
- "매번 주방 기름때 못 지우는 분들, 이거 아세요?"
- "옷을 드라이 맡기는 게 아깝다고 느끼는 분들."
```

**구조**: `[특정 문제/상황 겪는 분들]` + `"이게 원인/방법이에요"` + 개인화

---

### 패턴 5: 즉시성형 (지금 당장 확인)
> "지금 바로 확인해보세요" / "오늘 저녁에 꼭 해보세요"

**"지금 당장"** 행동해야 한다는 긴박감을 만든다.

```
예시:
- "냉장고 지금 열어서 이거 있으면 당장 옮기세요."
- "지금 빨래통에 흰 옷 있으면 이거 먼저 보세요."
- "오늘 저녁 요리 전에 꼭 알아야 할 주방 팁."
- "오늘 세탁하기 전에 30초만 보세요."
```

**구조**: `"지금/오늘"` + `[행동]` + `"전에/먼저"` + 행동 유도

---

### 패턴 6: 비교형 (전/후 or A vs B)
> "이것 vs 저것" / "하기 전/후 차이"

**결과의 차이**를 보여주겠다는 예고로 시청자의 기대감을 자극한다.

```
예시:
- "이렇게 보관하면 2배 오래가요 (전/후 비교)."
- "세제 한 스푼 vs 반 스푼, 세탁 결과 달라요."
- "냉장고 정리 전/후, 식비가 달라집니다."
- "흰 옷 세탁 전/후 — 이 한 가지 차이입니다."
```

**구조**: `[A 방법] vs [B 방법]` 또는 `"전/후"` + `[결과 차이 암시]`

## 5. Prompt 개선 방향

> ⚠️ 이 섹션은 `lib/openai.ts`의 `qualityBar`에 추가할 문구 **초안**이다.  
> **코드 수정 금지** — 검토 후 별도 atomic task로 적용.

### 5-1. 현재 hook 가이드 (lib/openai.ts line 279-281)

```
- Scene 1 (hook): 1~2초 안에 궁금증 자극. 시청자가 스크롤을 멈추는 질문/주장.
  ✅ "이거 아직도 이렇게 보관하세요?" / "대부분 이 실수 때문에 금방 상합니다"
  ❌ "오늘은 보관법을 알아볼게요" — 시청자 이탈
```

### 5-2. 강화 초안 (추가 삽입 위치: line 281 이후)

```
HOOK PATTERN ENFORCEMENT (scene 1 — CRITICAL):
Use one of these 6 proven hook patterns. Weak hook = viewer scrolls away in 1 second.

Pattern A (금지형): "[오브젝트]를 [흔한 방법]으로 하시는 분, 지금 당장 바꾸세요."
  ✅ "흰 옷 세탁기에 그냥 넣으면 안 됩니다, 이유 있어요."
  ✅ "달걀을 냉장고 문칸에 넣지 마세요."

Pattern B (손실형): "[행동] 잘못하면 [구체적 손해]가 생겨요."
  ✅ "세탁 후에도 냄새 안 빠지는 이유, 세제가 너무 많아서예요."
  ✅ "이 실수 하면 옷 수명이 절반 줄어요."

Pattern C (반전형): "[상식]이 사실은 [반대/틀림]입니다."
  ✅ "뜨거운 물로 흰 옷 세탁하면 안 됩니다, 더 노래져요."
  ✅ "냉장고 가득 채우면 오히려 더 빨리 상해요."

Pattern D (타깃형): "[특정 문제 겪는 상황]인 분들, 이게 원인이에요."
  ✅ "세탁 후에도 흰 옷이 계속 누런 분들, 이거 하나만 바꾸세요."

Pattern E (즉시성형): "지금 [오브젝트] 확인해보세요, 이거 있으면 [문제]입니다."
  ✅ "지금 빨래통에 흰 옷 있으면, 세탁 전에 이거 먼저 보세요."

  ❌ "X분이면 없어집니다" 형태의 결과 보장형 단독 사용 금지 (금지형/반전형과 결합 필요)
```
