## 1. 이야기 명확성 (20점)

| ID | 평가 항목 | 배점 | 내 점수 | 메모 |
|----|-----------|:----:|:-------:|------|
| hook_3sec | 첫 3초 후킹 — 시작 3초 내 궁금증 유발 여부 | 5 | /5 | |
| story_structure | 상황→갈등→반전→여운 4단계 구조 | 10 | /10 | |
| closing_line | 마지막 한 줄 여운 문장 (행동 유도 ❌) | 5 | /5 | |
| | **소계** | **20** | **/20** | |

**판단 기준 메모:**
- `hook_3sec` 5점: 첫 장면이 갈등/질문을 즉시 제시 / 3점: 배경은 좋으나 스토리 불명확 / 1점: 평범한 오브젝트 컷
- `story_structure` 10점: 4단계 완벽 / 7점: 3단계 명확 / 4점: 2단계 / 2점 이하: 나열식
- `closing_line` 5점: "…그래서 알게 됐어요" 류 감정 마무리 / 2점: 어색한 마무리 / 0점: 행동 유도 문구
