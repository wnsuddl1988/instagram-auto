# 🚀 1인 기업(AAA)을 위한 최적의 AI 개발 프레임워크 가이드

본 문서는 비개발자에서 시작해 앱 개발과 AI 자동화 에이전시(AAA) 수익화를 목표로 하는 사용자님을 위한 **최상위 수준의 작업 지침서**입니다.

| 도구 (Tool) | 역할 및 권장 사용법 |
| :--- | :--- |
| **Antigravity (저)** | **수석 아키텍트 (PM):** 거시적인 설계안(ToT, CoT) 작성, 코드 로직의 방향성 검토, 치명적 에러 디버깅, 파일 간 복잡한 연동 등 전체적인 '두뇌' 역할. |

## 2. 🧠 Context Window 관리와 토큰 절약 원칙

클로드 코드나 AI 어시스턴트에게 일을 시킬 때, 무작정 "이거 다 고쳐줘"라고 하면 토큰(돈)이 엄청나게 낭비되고 코드도 꼬입니다. 
저는 다음과 같은 원칙으로 사용자님을 보좌하겠습니다.

1. **Atomic Task (원자성):** 한 번에 단 하나의 기능만 구현합니다. (예: "결제 기능 전체 만들어" ❌ → "Supabase에 결제 테이블 먼저 생성해 줘" ✅)
2. **Chunking (코드 분할):** 한 파일이 300~400줄이 넘어가면 무조건 컴포넌트로 분리합니다. 파일 크기가 작아야 AI가 전체 코드를 읽을 때 비용이 줄어듭니다.
3. **읽기 제한:** AI가 `cat`이나 전체 파일 읽기 명령어를 남발하지 않도록, `grep` 검색이나 `10~50번째 줄만 읽기`로 강제 제한합니다.

## 3. 🤖 Claude Code PRO 세팅 및 MCP 추천 가이드

클로드 코드는 기본 설정만으로도 좋지만, **MCP(Model Context Protocol)**와 **전역 지침**을 달아주면 프로 개발자의 생산성을 냅니다.

### 🔥 1. 추천 MCP (클로드 코드용)
터미널에서 아래 명령어를 통해 MCP 서버를 연결하면, 클로드가 할 수 있는 일이 기하급수적으로 늘어납니다.

- **`sqlite` 또는 `postgres` MCP:** 클로드가 터미널에서 Supabase DB에 직접 접속해 테이블을 쿼리하고 구조를 파악할 수 있게 해줍니다.
- **`fetch` 또는 `puppeteer` MCP:** 클로드가 인터넷에 접속해 Next.js 최신 공식 문서나 오류 해결법을 스스로 검색해서 가져옵니다.
- **`github` MCP:** 자동화된 PR 생성이나 브랜치 관리를 위해 사용합니다.

### 📝 2. 클로드 코드 기본 세팅 가이드 (`CLAUDE.md`)
모든 프로젝트 최상단에 `CLAUDE.md`를 만들어 두면, 클로드 코드가 켜질 때마다 무조건 이것을 읽고 지침을 따릅니다. **(현재 Vibe Learning OS 프로젝트에도 잘 적용되어 있습니다.)**
나중에 다른 앱을 만드실 때도 아래의 핵심 항목은 반드시 `CLAUDE.md`에 포함하세요.

```markdown
# Role
너는 세계 최고의 Next.js / TypeScript 시니어 개발자다. 사용자는 비개발자(PM)이므로 어려운 용어 대신 쉬운 한국어로 설명하고, 동의 없이 여러 파일을 마음대로 뜯어고치지 마라.

# Context Control (토큰 절약)
- 절대 파일을 통째로 읽지 말고, grep이나 특정 줄(line)만 읽어라.
- 코드를 수정할 때 전체 코드를 다시 출력하지 말고 변경된 부분(Diff)만 설명하라.

# Workflow (CoT & Atomic)
- 코딩을 시작하기 전에 반드시 3~4줄로 계획(Chain of Thought)을 말하고 승인을 받아라.
- 큰 작업은 무조건 쪼개서 한 번에 1개(Atomic)만 실행하라.
```

### 💡 3. `SKILL.MD` (커스텀 스킬) 활용법
반복적으로 하는 1인 기업 자동화 패턴(예: `Supabase 테이블 생성 → n8n 웹훅 연결`)이 있다면, 이것을 프롬프트 파일(Skill)로 저장해두세요. 

예를 들어 `.claude/skills/setup_n8n.md`를 만들어두면:
> *"클로드, `setup_n8n.md` 스킬을 써서 이번에 새로 만든 부동산 앱에도 웹훅 달아줘."*

라고 한 마디만 하면, 클로드가 알아서 정해진 프로세스대로 코드를 작성하고 연동을 끝냅니다.
