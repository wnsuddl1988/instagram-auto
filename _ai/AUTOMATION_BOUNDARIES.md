# Automation Boundaries

## Allowed Automation

HANDOFF/report/review/NEXT_ACTION/PROJECT_STATE/TRACKING draft 작성, targeted test, diff summary, commit message draft, Obsidian template update

## Forbidden Without Explicit Approval

commit, push, deploy, DB schema/migration, env/secret, dependency, external sending, payment/purchase/sale/order, investment/trading execution, legal execution, account/customer/production data changes
